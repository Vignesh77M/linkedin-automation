# LinkedIn Automation POC (Go + Rod)

This project is a proof-of-concept demonstrating advanced browser automation using Go and Rod.

## Features
- Credential-based login
- Human-like typing and delays
- Browser fingerprint masking
- Profile search & connection requests
- SQLite-based state tracking

## Usage
```bash
set LI_USERNAME="your_email"
set LI_PASSWORD="your_password"
go run cmd/main.go
